## The Data Stack (All Free/Public)

### Layer 1: GEMI Public API (FREE)
- **URL**: https://opendata-api.businessportal.gr/opendata/docs/
- **Auth**: Register for API key (free), test key: `api-docs-key`
- **Data**: Company name, AFM, GEMI number, legal form, incorporation date, status, capital, activities (KAD codes), management, contact details, financial data, website/e-shop
- **Search by**: AFM (ΑΦΜ), GEMI number, or company name
- **Rate limits**: Limited per IP on test key, production key via application

### Layer 2: Connector Graph (EXISTING)
- 9,661 companies already in the graph
- 5,320 people with relationship data
- 399 referral chains = trust network mapping
- Tags, sectors, connection strength

### Layer 3: Public Web Scraping
- Company website → tech stack, hiring pages, recent news
- Google Maps → reviews, photos, hours, rating
- Social media presence → LinkedIn company page, Facebook, Instagram
- News mentions → press coverage, regulatory issues
- Job postings → growth signals (hiring = expanding)

### Layer 4: Public Financial Data
- Published financial statements (via GEMI)
- Tax filing status (via TAXIS public endpoints)
- Court judgments / bankruptcies (via public registries)
- EU funding received (via ESPA/anaptyxi.gov.gr public lists)

### Layer 5: AI Analysis (THE MOAT)
- **Competitive positioning**: Where does this company sit vs. peers in the same KAD code?
- **Risk signals**: Declining revenue + hiring freeze + negative reviews = warning
- **Growth signals**: Revenue up + new job postings + EU funding = opportunity
- **Relationship map**: Who in YOUR network knows someone at this company? (Connector)
- **Actionable brief**: "Here's what this company needs. Here's how to approach them."

## Technical POC Spec

### `company_brief.py`

```python
# Pseudo-architecture

class CompanyBrief:
    def __init__(self, afm: str):
        self.afm = afm
    
    async def generate(self) -> BriefReport:
        # 1. GEMI API — core company data
        gemi_data = await self.fetch_gemi(self.afm)
        
        # 2. Connector — network enrichment
        connector_data = await self.search_connector(gemi_data.name)
        
        # 3. Web scraping — digital presence
        web_data = await self.scrape_web(gemi_data.website, gemi_data.name)
        
        # 4. Peer comparison — same KAD code companies
        peers = await self.find_peers(gemi_data.kad_code)
        
        # 5. AI synthesis — the magic
        brief = await self.ai_analyze(
            company=gemi_data,
            network=connector_data,
            web=web_data,
            peers=peers
        )
        
        return brief
```

### Dependencies
- `httpx` — async HTTP for GEMI API
- `beautifulsoup4` — web scraping
- `anthropic` — Claude API for AI analysis
- LOSC Connector API — network enrichment
- `weasyprint` or `jinja2` — HTML/PDF output

### GEMI API Endpoints (from Swagger)

- `GET /company/search?name={name}` — search by name
- `GET /company/{gemi_number}/financials` — financial statements
- `GET /company/{gemi_number}/management` — board/management
